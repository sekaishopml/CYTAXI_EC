# ADR: Cobalt theme (Hallmark) migration

**Date**: 2026-07-20

**Context**: Original UI used green identity (`#006c49`) with glassmorphism. IA-generated code had anti-patterns.

**Decision**: Apply Hallmark skill Cobalt theme:
- Accent: `#3b82f6`
- Paper: `#f5f6f8`
- Fonts: Space Grotesk (display), Inter (body), JetBrains Mono (labels)
- Radii: 6px / 10px
- No glassmorphism; hairline borders
- Easing: `cubic-bezier(0.16,1,0.3,1)`

**Status**: Applied
